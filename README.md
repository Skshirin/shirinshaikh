# 🌐 My Portfolio

A modern and responsive developer portfolio built using Next.js, showcasing my projects, skills, and experience as a web developer.

## 🚀 Live Website

👉 https://shirin-portfolio.netlify.app/

## ✨ Features

* 💼 Showcases my projects and work
* 🧑‍💻 Clean and modern UI
* 📱 Fully responsive design
* ⚡ Fast performance with Next.js
* 📬 Contact section for easy reach

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* TypeScript

## 📂 Project Structure

* `/app` – Main application pages
* `/components` – Reusable UI components
* `/public` – Static assets

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Skshirin/shirin-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
